# Snake-Game

A browser port of the classic Nokia phone game.

<br>

https://github.com/user-attachments/assets/a353ecc3-8acb-4481-80a5-df97b486ddb0

<br>

## Requirements

```
node
npm
```

<br>

## Install node modules

```sh
npm i
```

<br>

## Run the dev server

```sh
npm run dev
```

<br>

## Build the dist folder for deployment

```sh
npm run build
```
