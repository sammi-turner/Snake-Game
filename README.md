# Snake-Game

A browser port of the classic Nokia phone game.

<br>

https://github.com/user-attachments/assets/7d2ec307-c19c-41c7-8846-854d3db98870

<br>

## Requirements

```
node
npm
```

<br>

## Install node modules

```
npm i
```

<br>

## Run the dev server

```
npm run dev
```

<br>

## Build the dist folder for deployment

```
npm run build
```
